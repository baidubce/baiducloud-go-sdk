# Baidu Cloud V2.0 SDK for Go

## 要求
- 您需要确保本地安装的 go 环境版本不低于 1.11.0.

## 安装
你可以使用 `go mod` 来管理你的依赖，
```sh
  go get github.com/baidubce/baiducloud-go-sdk
```
